Voice User Interfaces

This section will allow you to interact with the voice user interface/chatbot component of the project. It is best to use Google Chrome in a non-incognito window and have any adblock turned off. This is to minimise any interference with the mic input. This is an embedded version of Dialogflow's web demo and isn't perfect.

Consider this chatbot as a device you might have physically in your home. It allows you to make energy trades and other queries involving energy.

Type any statements or questions by clicking in the text box, typing and pressing ENTER. To use voice: Click the microphone button. It will turn orange. Speak your question or queury. Now press it again. It should recognise your speech.
